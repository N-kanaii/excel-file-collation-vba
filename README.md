# excel-file-collation-vba
業務で発生する複数ファイルの集約作業を自動化するExcel VBAツールです。  指定フォルダ内の複数Excelファイル（パターン一致）を順次読み込み、 データを1つのワークブックへ統合します。  手作業で行っていたデータ集約業務を効率化することを目的に開発し、 公開用に機密情報を除去したサンプルコードとして整理しています。


Excel File Collation Tool (VBA)
概要

複数のExcelファイルを自動で読み込み、1つの集約シートに統合するVBAツールです。
指定フォルダ内の対象ファイルをパターン指定で取得し、データを一括処理します。

業務で発生するファイル集約作業の効率化を目的として作成し、
公開用に機密情報を除去・処理を簡素化しています。

主な機能
指定フォルダ内の複数Excelファイルを一括取得
ファイル名パターン（例: keyword_*.xlsx）による抽出
各ファイルのデータを順次読み込み
集約用シートへデータを統合
元ファイルは保存せずにクローズ（安全設計）
処理の流れ
指定フォルダ内の対象ファイルを取得
各ファイルを順番に開く
データを取得（UsedRange）
集約シートへ追記
ファイルを保存せずに閉じる
全ファイル処理完了までループ
使用技術
Excel VBA
Workbook / Worksheet 操作
Dir関数によるファイル取得
ループ処理
バッチ処理設計
実行方法
本リポジトリをダウンロード
.xlsm ファイルを開く
マクロを有効化
collatedata マクロを実行


ポイント
